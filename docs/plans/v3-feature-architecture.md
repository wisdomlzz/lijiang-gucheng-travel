# 丽江古城游 V3.0 功能架构方案

> 基于功能清单0628v0.1，去掉5A数据上报，按产品完整性梳理。

## 已确认的产品决策

1. **商户端共用 C 端框架**，按 `user.role` 动态展示宫格/页面
2. **商户可自助维护店铺信息**，提交后 PC 端审核生效
3. **一键服务共享表单**，按角色过滤服务类型（游客：行李搬运/送货；商户：垃圾清运×2/布草/送水/送货）
4. **积分体系游客+商户共享**，纳西人打卡两端都做
5. **商城外链 CRMEB**，不内置交易闭环；积分兑换也跳外链
6. **AI 三种能力都模拟**（知识库问答/数据库问数/文档合规审查）
7. **人流量预警完整模拟**（PC 管理 + C 端地图色块联动）
8. **志愿者签到签退**只做时间水印+拍照，不做真实 GPS
9. **院落预约完整流程**（选时段→预约码→核销）
10. **宣传素材 + 档案管理 → 合并**为"宣传素材管理"
11. **智慧营销不单独开模块**——线上预约由内容管理覆盖，一键服务由便民服务覆盖
12. **讲解服务 C 端 Demo 暂不做**
