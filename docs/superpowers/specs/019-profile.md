# 个人中心 — 功能需求规格

> **话题标签:** `019-profile`
> **上次更新:** 2026-07-05

## 业务定位

C 端"我的"页面，聚合个人相关信息：头像、角色标签、积分、志愿服务统计、地址、收藏、设置等。

**本质：** 用户个人信息聚合页 + 各功能入口。

## 页面清单

| 端 | 页面 | 路由 | 说明 |
|---|---|---|---|
| C | ProfilePage | `/c/profile` | 个人中心聚合页 |

## 聚合内容

- 用户信息（姓名、角色、手机号）
- 积分余额 + 入口
- 收藏夹 + 入口
- 地址管理 + 入口
- 志愿服务统计（累计时长、活动次数）
- 我的投诉 + 入口
- 我的发布 + 入口
- 关于/登录信息

**约束：** 纯展示 + 路由入口，无编辑个人信息功能。

## 本 Demo 的范围

- ✅ **C 端**: ProfilePage — 展示用户信息（姓名/手机号/角色标签）、积分入口、收藏入口、地址入口、志愿服务统计、投诉入口、关于/登录信息
- ⚠️ **纯展示**: 仅聚合各功能路由入口 + 展示统计数据
- ❌ **无编辑功能**: 不支持修改头像/昵称/手机号等个人信息
- ❌ **无独立 store**: 用户数据来自 auth-store（shared/stores/auth-store.ts）
