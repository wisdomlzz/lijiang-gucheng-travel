# 个人中心（Profile）

## 产品定位与边界

个人中心是 C 端用户的"我的"页面，提供用户身份展示、积分概览、志愿服务统计、功能快捷入口（收藏/发布/随手拍）、订单入口、设置菜单及退出登录。

边界：
- 仅限 C 端（游客端）访问
- 不包含账号编辑、实名认证、支付等深度用户管理
- 头像/昵称为固定 Mock 数据

## 核心用户角色

| 角色 | 描述 |
|------|------|
| 游客（tourist） | 查看个人信息，进入各功能模块 |

## 核心业务流程

1. 用户进入个人中心
2. 顶部展示用户头像、昵称、ID（Mock 数据）
3. 积分卡片展示当前积分，点击进入积分中心
4. 如已认证志愿者，展示志愿服务统计（累计时长、参与场次）
5. 功能快捷区：我的收藏、我的发布、随手拍
6. 便民服务订单快捷入口，显示进行中订单数量
7. 菜单列表：积分中心/我的预约/纳西人打卡/我的店铺/收货地址/消息通知/我的投诉
8. 供应商入驻入口
9. 退出登录按钮

## 核心数据模型

Profile 页面自身无独立数据模型，依赖以下 store：
- `useAuthStore` — 用户身份信息
- `usePointsStore` — 积分余额
- `useConvenienceStore` — 订单数据
- `useVolunteerStore` — 志愿服务记录

- ID："LJ88392156"
- 头像：固定图片资源
