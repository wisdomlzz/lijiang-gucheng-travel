# 便民信息（Info）

## 产品定位与边界

便民信息模块是丽江古城 C 端的信息公告板，提供"公房公告"、"房屋信息"、"举贤纳仕"、"其它"四个类别的资讯浏览，以及"古城资讯"综合资讯页。支持用户发布信息。

边界：
- C 端（游客端）浏览 + 发布
- 资讯数据复用 content feature 的新闻 store（`useContentNewsStore`）
- 用户发布功能为模拟 UI，未对接真实提交逻辑

## 核心用户角色

| 角色 | 描述 |
|------|------|
| 游客（tourist） | 浏览资讯、发布便民信息 |

## 核心业务流程

**资讯浏览（正常流程）**
1. 用户进入便民信息页
2. 选择分类 Tab（公房公告/房屋信息/举贤纳仕/其它）
3. 关键词搜索标题
4. 点击卡片进入详情页
5. 详情页展示：大图横幅、标题卡片、详情图文

**用户发布**
1. 用户点击"我要发布"FAB 按钮
2. 跳转发布页（未实现，路由 `/c/info/create` 可能不存在）

**我的发布**
1. 用户从个人中心进入"我的发布"
2. 查看已发布信息列表（审核中/已发布/已拒绝）
3. 支持删除操作和重新提交

## 核心数据模型

复用 content feature 的 `NewsItem`：

```typescript
type NewsCategory = "公房公告" | "房屋信息" | "举贤纳仕" | "其它"

interface NewsItem {
  id: string
  imageUrl: string
  title: string
  tag: string
  tagColor: string
  date: string
  summary: string
  category: NewsCategory
  heroTitle?: string
  body?: string[]
  subImage?: string
}
```
