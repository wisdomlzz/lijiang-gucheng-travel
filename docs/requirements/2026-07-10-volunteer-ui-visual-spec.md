#### 1.6 数据字段

| 展示区域 | 字段（中文） | 字段名 | 来源 | 说明 |
|----------|-------------|--------|------|------|
| Tab | 选中状态 | activeTab | 前端状态 | "all" / "active" / "ended" |
| 卡片 | 标题 | title | VolunteerActivity | - |
| 卡片 | 开始时间 | startTime | 同上 | 格式: MM-dd HH:mm |
| 卡片 | 结束时间 | endTime | 同上 | 格式: MM-dd HH:mm |
| 卡片 | 地点 | location | 同上 | - |
| 卡片 | 已报名数 | enrolledCount | 前端计算 | signUps.filter(activityId).length |
| 卡片 | 人数上限 | maxParticipants | VolunteerActivity | - |
| 卡片 | 状态 | status | 同上 | draft/published/in_progress/ended/cancelled |
| 卡片 | 报名开始时间 | enrollStartTime | 同上 | 用于判断"报名未开始" |
| 卡片 | 报名截止时间 | signUpDeadline | 同上 | 用于判断"报名已截止" |
| FAB | 显示条件 | - | auth store | 仅认证志愿者且页面非空时显示 |

```
│  └─────────────────────────────────────────────────────────┘  │
│                                                               │
│  ⚠️ 驳回原因不能为空                                          │
│                                                               │
│  ┌──────────────┐   ┌────────────────────────┐              │
│  │    取消      │   │        驳回            │  ← 需loading │
│  └──────────────┘   └────────────────────────┘              │
└─────────────────────────────────────────────────────────────┘
```

#### 5.6 对话框通用规范

| 行为 | 要求 |
|------|------|
| 确认按钮 | 必须有 `disabled={loading}` + 显示 spinner/文字变化 |
| 取消按钮 | loading 时同时禁用，防止对话框意外关闭 |
| 必填校验 | 提交时校验，未填项下方显示红色提示文字 |
| 错误反馈 | API 错误用 `toast.error()` 显示 |
| ESC 关闭 | 按 ESC 不提交，仅关闭 |
| 点击蒙层 | 点击背景不提交，仅关闭 |
