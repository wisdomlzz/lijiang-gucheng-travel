# 遗产知识（Heritage）

## 产品定位与边界

遗产知识模块是丽江古城文化遗产的数字百科，面向游客提供古城内 8 类文化遗产（道路、水系、井/泉、古桥、古树、保护民居、历史建筑、人文环境）的浏览、搜索、定位导航服务。只读展示，不涉及编辑/管理。

边界：
- 仅限 C 端（游客端）访问
- 不包含商户、POI、路线等内容的遗产关联
- 不含后台管理系统

## 核心用户角色

| 角色 | 描述 |
|------|------|
| 游客（tourist） | 浏览遗产知识，查看详情、位置、导航 |

## 核心业务流程

1. 游客进入遗产知识列表页
2. 按分类 Tab（全部/8 种类型/离我最近）筛选
3. 关键词搜索（名称/描述）
4. 点击卡片进入详情页
5. 详情页展示：图片轮播、分类标签、基本信息、位置地图、图纸资料、文字介绍
6. 支持全屏图片浏览、跳转高德导航

## 核心数据模型

```typescript
type HeritageType = "road" | "water" | "well" | "bridge" | "ancient-tree" | "protected-house" | "historic-building" | "human-environment"

interface HeritageItem {
  id: string
  type: HeritageType
  name: string
  area: Area  // "大研" | "白沙" | "束河" | ...
  location: { lat: number; lng: number }
  description: string
  photos: string[]
  address?: string
  preservationStatus?: string
  basicInfo?: string
  drawings?: string[]
  heritageSubType?: string
  extra?: Record<string, string | string[] | boolean>  // 类型特有字段
}
```

种子数据：8 种类型各约 4-6 条示例数据，定义在 `shared/data/` 下各类型文件中。
