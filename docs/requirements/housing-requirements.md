# 公房服务（Housing）

## 产品定位与边界

公房服务模块展示丽江古城国有公房的租赁信息，面向游客和商户提供公房列表浏览、详情查询（含基础信息、合同信息、租赁信息）及全景预览功能。

边界：
- 仅限 C 端（游客端）访问
- 仅展示功能，不含租赁申请、合同签署、支付等交易流程
- 不含后台管理系统

## 核心用户角色

| 角色 | 描述 |
|------|------|
| 游客/商户（tourist/supplier） | 浏览公房列表，查看详情与全景 |

## 核心业务流程

1. 用户进入公房列表页
2. 查看全部/出租/未出租筛选 + 关键词搜索
3. 点击卡片进入详情页
4. 详情页展示：照片轮播、基础信息、合同信息、租赁信息三大分组
5. 支持全景漫游（模拟 720° 全景，进度条播放）

## 核心数据模型

```typescript
interface House {
  id: number
  name: string           // 公房名称，如"光义街9号公房"
  addr: string           // 完整地址
  status: "rented" | "idle"  // 出租/未出租
  statusText: string
  area: string
  areaName: string
  hasPano: boolean       // 是否有全景
  meta: string[]         // 摘要标签：[面积, 商业类型, 结构]
  photos: { bg: string }[]  // 背景渐变图（占位）
  basic: Field[]         // 基础信息
  contract: Field[]      // 合同信息
  lease: Field[]         // 租赁信息
}
```

种子数据：5 条固定真实样例 + 50 条随机生成数据，总计 55 条。
