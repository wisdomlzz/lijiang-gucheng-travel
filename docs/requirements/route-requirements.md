# 游玩攻略（Route）

## 产品定位与边界

游玩攻略模块为丽江古城游客提供精选游览路线的浏览、详情查看和模拟导航预览体验。包含路线列表、攻略详情（含视频、图文混排内容块）、路线示意地图和分站导航模拟。

边界：
- 仅限 C 端（游客端）访问
- 路线数据复用 content feature 的 guide store（`useContentGuideStore`）
- 导航为前端模拟，不调用真实地图导航 SDK
- 不含后台路线管理页面

## 核心用户角色

| 角色 | 描述 |
|------|------|
| 游客（tourist） | 浏览攻略路线，查看详情和模拟导航 |

## 核心业务流程

**路线浏览**
1. 用户进入游玩攻略列表页
2. 关键词搜索路线名称/景点名
3. 点击卡片进入路线详情页

**路线详情**
1. 展示封面大图
2. 可选攻略视频播放（支持视频组件）
3. 路线基本信息（时长/距离/景点数）
4. 错峰建议（模拟时段拥挤度）
5. 路线示意图片
6. 攻略详情内容块（图文/视频混排）

**路线预览（模拟导航）**
1. 进入预览模式，全屏地图背景
2. 路线路径 SVG 绘制 + 景点标记
3. 底部卡片显示当前站点
4. 上一站/下一站切换
5. 导航按钮（Mock，Toast 反馈）
6. 游览进度条

## 核心数据模型

```typescript
interface TravelGuide {
  id: string
  name: string
  tags: string[]
  duration: string
  difficulty: string
  stops: number       // 景点数
  distance: string    // 路线长度
  spotNames: string[] // 途经景点名列表
  description: string
  cover: string
  spots: RouteSpot[]  // 景点坐标（百分比定位）
  hasVideo?: boolean
  videoUrl?: string
  videoCoverUrl?: string
  videoDuration?: string
  contentBlocks?: ContentBlock[]  // 图文/视频混排内容
}

interface RouteSpot {
  id: string
  name: string
  desc: string
  top: string   // 百分比定位
  left: string  // 百分比定位
}
```
